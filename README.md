## wx-devmode-advance
1. 图文消息
2. 获取access_token
3. 回复图片消息
4. 回复音乐消息
5. 自定义菜单
6. 百度翻译

## wx-auth
1. 直接拉取微信用户信息进行登录
2. 把自己的账户体系和微信账户进行绑定
3. unionid

**注意：**
1. 组装授权地址的时候需要注意参数的顺序跟api中是否保持一致
2. 回调地址必须是在公网能够访问到的
3. 回调地址域名的设置
